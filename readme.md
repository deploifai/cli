# Deploifai CLI

### Installation

```shell
pip install deploifai-cli
```

### Usage

```shell
# ensure the cli is accessible
which deploifai
# show help message
deploifai --help
# show version
deploifai --version
```

## Contribute

### Setup Development Environment

```shell
python -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -r requirements.txt
```

Install `deploifai` cli tool for testing.

```shell
pip install --editable . 
```
